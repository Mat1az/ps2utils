## Key Features

* PS2 Manager:
  - FMCB/Funtuna Settings
  - Folders/Fixes
  - Updates
* Game Manager:
  - ISO/ZSO Info
* POPS Manager:
  - CUE -> VCD
* OPL Manager
  - APPS Installer
  - Game Config
* SaveData Manager
  - Conversion Tool
